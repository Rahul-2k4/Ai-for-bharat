# Demo Script

## Time budget

Target: 4 to 6 minutes.

## Flow

1. Problem statement (30-45 sec)
2. Why current alternatives fail (30 sec)
3. Live product walkthrough (2-3 min)
4. Technical differentiators (45-60 sec)
5. Measured outcomes + next steps (30-45 sec)

## Demo proof checklist

- [ ] Fresh run from documented setup steps.
- [ ] No hidden manual prep.
- [ ] Screenshots/video in `assets/` match current build.
- [ ] README reflects exact demo behavior.
